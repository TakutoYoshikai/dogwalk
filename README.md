# dogwalk
dog walking animation

### Requirements
* npm

### Usage
**install**
```bash
npm install TakutoYoshikai/dogwalk
```

**import library**
```html
<script src="./node_modules/dogwalk/index.js"></script>
```

**name class**
```html
<p class="dogwalk">Hello World.</p>
<p class="dogwalk">I love dogs.</p>
```

**run DogWalk**
```html
<script>
  DogWalk();
</script>
```
